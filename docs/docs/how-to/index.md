# How-to Guides

How-to guides are **problem-oriented** recipes that help you accomplish specific tasks. Unlike tutorials, they assume you already understand the basics and need to solve a particular problem.

## Available Guides

### Finding Paths

- [Find the Shortest Path](./find-shortest-path) - Get from A to B efficiently
- [Find All Possible Routes](./find-all-routes) - Discover alternative paths

### Analyzing Importance

- [Find Influencers](./find-influencers) - Identify the most important nodes
- [Find Bridge Nodes](./find-bridge-nodes) - Locate critical connection points

### Dependency Analysis

- [Detect Circular Dependencies](./detect-circular-dependencies) - Find cycles in your graph
- [Analyze Impact](./analyze-impact) - Understand what's affected by changes

### Data Export

- [Export for Visualization](./export-for-visualization) - Create visual diagrams with D3.js, Gephi

### Filtering and Querying

- [Filter Nodes by Attributes](./filter-by-attributes) - Find specific subsets
- [Extract Subgraphs](./extract-subgraphs) - Create focused views of your data
